Kai Series

By digital on Mar 23, 2013

「改」Series (a.k.a. Kai), is a series of the missing child themes for WordPress default theme Twenty Twelve, Twenty Eleven, and Twenty Ten. The suffix “Kai” of this series is inspired from Japanese anime ドラゴンボール改 (a.k.a. Dragon Ball Kai).

Features

  • Written in Sass (Kai 12) and Less (others), no IE support
  • Better font stack for Chinese, Japanese and other asian languages
  • Better code snippets style
  • Adds .well block inspired by Bootstrap
  • Fix media query issue
  • Theme updater support
